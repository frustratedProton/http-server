# Security Policy

## Project Disclaimer

This is a **educational project** and is not intended for production use.

## Supported Versions

Because this is a hobby project, I only support the code currently on the main branch.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

**Please DO NOT open a public issue for security vulnerabilities.**

If you have found any vulnerability, please report it privately so I can learn from it and fix it safely.

### Contact Information

1. **GitHub Private Reporting**: Use the "Report a vulnerability" button in the Security tab
2. **Email**: [aryansingh29052003@gmail.com]

### Please Include:
- Description of the vulnerability
- Steps to reproduce
- Affected files/endpoints

### Expectations:
- **Response**: Within 48 hours
- **Updates**: I'll keep you informed of progress
- **Credit**: Happy to acknowledge your contribution

### Bounties: This is an open-source learning project. I do not offer bug bounties or financial rewards.

Thank you for helping improve this project!
